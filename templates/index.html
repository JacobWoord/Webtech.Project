Wat is HTML & CSS?

<HTML>

HyperText Markup Language - De structuur van een website. HTML bepaalt wat er op een pagina staat: tekst, afbeeldingen, links, formulieren, etc.

"HTML is het skelet van het web"

{ CSS }

Cascading Style Sheets - De styling van een website. CSS bepaalt hoe alles eruitziet: kleuren, fonts, layout, animaties, etc.

"CSS is de kleding van het web"

Pro Tips & Best Practices

            

                

                    
‚úÖ DO: Gebruik Semantische HTML

                    
<header>
  <nav>...</nav>
</header>
<main>
  <article>...</article>
</main>

                    
Waarom? Betere SEO en accessibility!

                


                

                    
‚ùå DON'T: Div Soup

                    
<div class="header">
  <div class="nav">...</div>
</div>
<div class="main">...</div>

                    
Waarom niet? Geen semantische betekenis!

                


                

                    
‚úÖ DO: CSS Variables

                    
:root {
  --primary: #3498db;
}
.button {
  background: var(--primary);
}

                    
Waarom? Makkelijk aanpasbaar!

                


                

                    
‚ùå DON'T: Te veel !important

                    
.text {
  color: red !important;
  font-size: 16px !important;
}

                    
Waarom niet? Moeilijk te overschrijven!
